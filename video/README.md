# Warden LinkedIn Video

Remotion project for a cinematic LinkedIn post video about Warden.

## Quick Start

```bash
cd video
npm install
npm run studio
```

## Render

```bash
cd video
npm run render
```

Output:

- `video/out/warden-linkedin-cinematic.mp4`

## Story Flow

1. Hook: synthetic macOS terminal typing + live output
2. The Council: Watchman, Engineer, Diplomat
3. Coverage: SAST + DAST modes
4. Close: autonomous loop and human-in-the-loop merge control
